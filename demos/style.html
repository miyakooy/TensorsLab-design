TensorsLab visual style board

This board defines a more product-grade visual language for TensorsLab: a blue-led interface tone for clarity, green accents for trust and confirmation, and generous white space to keep the layout readable. It is polished enough to act as a usable visual reference for homepage, launch, and deck design.
